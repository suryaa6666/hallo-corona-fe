
# Hallo Corona

Simple project made with react js to read article about corona and live consultation with doctor.
## Run Locally

Clone the project

```bash
  git clone https://github.com/suryaa6666/hallo-corona-fe
```

Go to the project directory

```bash
  cd hallo-corona-fe
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
